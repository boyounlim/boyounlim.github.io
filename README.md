# GitHub Pages 개발자 포트폴리오 (v1)

순수 `HTML/CSS/JS`로 만든 원페이지 개발자 포트폴리오 템플릿입니다.

## 포함된 섹션

- 소개 (`#about`)
- 기술 (`#skills`)
- 프로젝트 (`#projects`)
- 경력 요약 (`#experience`)
- 연락처 (`#contact`)

## 로컬 확인

`index.html`을 브라우저에서 열어 바로 확인할 수 있습니다.

## GitHub Pages 배포 (User Pages)

1. GitHub에 `YOUR_GITHUB_ID.github.io` 이름으로 저장소를 생성합니다.
2. 이 폴더 파일 전체를 `main` 브랜치 루트에 push 합니다.
3. GitHub 저장소 `Settings > Pages`에서:
   - `Build and deployment`의 `Source`를 `Deploy from a branch`
   - `Branch`를 `main` / `/ (root)`로 설정
4. 배포 완료 후 `https://YOUR_GITHUB_ID.github.io`로 접속합니다.

## 바꿔야 할 플레이스홀더

- `index.html`의 이름/소개 문구
- 프로젝트 카드 3개의 `Demo`, `GitHub` 링크
- 연락처의 이메일, GitHub ID, LinkedIn URL
- 이력서 버튼 링크 (`href="#"`)
